<p>

In terms of target audience, I delved into the topic by creating personas and researching in order to understand the user.

</p>

<p>

Once the research was completed, I started out by creating an in-depth sitemap of an already existing website in order to understand what components already exist. I considered the different elements they had including cards, banner types, form inputs.

</p>

<p>

I began to create the brand around AthenaXNova. I developed the colour palette, typography, and logo. The visual design came into play when I began to pick the type scale, and designed some sample components that would be featured on the website. I also began to consider accessibility and performance for the website itself.

</p>

<p>

The wireframes were the next step. I created wireframes for the homepage, product list page, product details page, and the checkout form page. They were high-fidelity and created in Adobe XD.

</p>

<p>

The development process began by the introduction of a pattern library. The pattern library is a collection of all the website components organized into sections. From the set-up of Jekyll—a static website generator—we were able to create short codes which would eventually be used to generate the website page. By using this website generator, it allowed for a more efficient development process and helped reduce any risks of duplicated code.

</p>

  • • writing in Markdown and YAML files

  • • preparing product data

  • • handing off work to teammates

  • • performance, accessibility, and browser testing
  • • using Terminal
  • • using the Ruby programming language

</li>

<p>

We covered a broad range of skills throughout the development of the pattern library. Some skills I developed include being able to develop my written, spoken, and visual abilities in a collaborative setting.

Overall, I am very pleased with the outcome of my e-commerce website. I was able to design and develop the AthenaXNova brand using the process of pattern libraries.

This project taught me how a complex process can be seamlessly combined into one system that can aid in simplifying a topic as complex as coding. The pattern library is a system that I will continue to use and it has enhanced my knowledge and skillset.

</p>
